# Haemoplasma_sloth_infections

This repository provides the dataset, R command lines and scripts used for the statistical analyses in the manuscript 'XXX' by XXX et al. In this study, we used data from veterinary evaluations conducted in wild brown-throated sloths (*Bradypus tridactylus*) and Linnaeus’s two-toed sloths (*Choloepus didactylus*)to investigate the effect of infection with haemotropic mycoplasmas (haemoplasmas) in French Guiana.

R command lines and scripts are available here : https://github.com/olivierduron/Haemoplasma_sloth_infections/blob/main/R_command_lines_and_scripts_haemoplasma_sloth.md

The dataset is available here : https://github.com/olivierduron/Haemoplasma_sloth_infections/blob/main/data_haemoplasma_sloth.xlsx
